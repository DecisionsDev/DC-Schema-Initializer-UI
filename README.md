# ODM-Schema Initializer
## Features
This is a source code with runnable jar which intialize ODM Decision center database by internally calling Decision center REST APIs.

## Requirements
 1. Decision center application should be up and running in new schema.
 2. JDK version 17 or up

## Deployment and Run
 1. Import the project source code
 2. Build the war file with maven or Directly Download the war file DC-Schema-Initializer-UI.war from repository and use it.
 4. Deploy or Run the war file: java -jar DC-Schema-Initializer-UI.war
 5. Application will run in the port 8080
